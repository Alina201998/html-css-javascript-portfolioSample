
## Personal Portfolio Page (HTML & CSS)

A simple personal portfolio webpage built using **HTML and CSS**.
This project was created as an early web development exercise to practice layout,
styling, and structuring a personal-style webpage.

---

## Features

- Clean and modern layout
- Profile section with image and introduction
- Skills section displayed using CSS Grid
- Projects section with descriptions
- Hover effects for interactive UI elements
- Fully responsive layout using Flexbox and Grid

---

## Technologies Used

- HTML5
- CSS3
